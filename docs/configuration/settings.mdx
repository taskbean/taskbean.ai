---
title: "Settings"
description: "Configure taskbean's behavior, model, theme, and more."
---

Access via the gear icon. Settings persist to `~/.taskbean/config.json`.

| Setting | Description |
|---------|-------------|
| **Model** | Select default model |
| **Timezone** | For time-aware reminders |
| **Theme** | Visual appearance |
| **Run on startup** | Register as Windows startup app |
| **Notifications** | Enable/disable toast notifications |
| **Model permission** | "ask" (confirm downloads) or "yolo" (auto-download best model) |
