---
title: "Nerd Mode"
description: "View telemetry, traces, metrics, and logs."
---

Click the **✨ Nerd** button to open the telemetry panel. Four tabs:

| Tab | Description |
|-----|-------------|
| **Events** | OTel span events with filter chips and search |
| **Metrics** | CPU, RAM, GPU, NPU sparkline cards |
| **Traces** | Waterfall trace view (proxied from Jaeger) |
| **Logs** | Severity-filtered log records |
