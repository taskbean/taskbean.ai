## Required

- **Windows 10 or 11**
- **Python 3.10+** — [python.org](https://www.python.org/downloads/)
- **Foundry Local** — `winget install Microsoft.FoundryLocal`

## Optional

- **Docker** — for Jaeger tracing (nerd mode)
- **Node.js 18+** — only needed for the legacy Node backend or Playwright E2E tests
