Quick Start

Install once. Link your agent. Done.

Taskbean is a CLI plus a local app. Your agent logs its own work via the Agent Skill manifest — you just live your life.

01 Install the CLI

# npm (macOS, Linux, Windows)
npm install -g taskbean
# macOS / Linux
curl -fsSL https://taskbean.ai/install | bash
# Windows PowerShell
iwr -useb https://taskbean.ai/install.ps1 | iex

02 Link your agent

bean install --agent copilot

Drops a SKILL.md into your agent's skills directory. Works with Copilot CLI, Claude Code, OpenCode, and Codex.

03 Let it log itself

bean add "Ship landing page"
bean done 42 --note "Merged #318"
bean report --today

Your agent uses these as it works. At EOD, bean report writes itself.

Works Offline

Installable as a PWA with full offline support. No internet required, ever. Your tasks are always accessible.

Natural Language AI

Talk to the bean. Say "add a task to review PRs by Friday" or "plan my day" and the AI handles the rest. Multi-turn conversations with full tool-calling — not just a chatbot skin on a text field.

Private by Default

Zero data leaves your device. No accounts. No telemetry phoning home. The AI runs on your hardware through Foundry Local.

Hardware Accelerated

Runs on your NPU, GPU, or CPU — whatever your machine has. Automatically selects the fastest execution provider available.

Multi-Model

Switch between Phi-4, Qwen, and other models on the fly. The model picker shows compatibility, size, and performance — hot-swap without restarting.

Smart Reminders

Windows toast notifications, snooze, overdue alerts, and recurring schedules. "Remind me to stretch every 45 minutes."

File Extraction

Drop meeting notes, emails, or PDFs — the AI extracts tasks automatically. Paste anything, and it pulls out the action items.

Nerd Mode

Live telemetry panel with CPU, RAM, GPU, NPU stats. OpenTelemetry traces, metrics sparklines, and Jaeger integration. See every token.
